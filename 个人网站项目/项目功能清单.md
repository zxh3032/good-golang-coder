# 个人网站 / 博客系统 — 功能清单

## 一、基础功能（MVP）

### 1. 用户与权限管理  
- 用户注册 / 登录（账号 + 密码）  
- 第三方登录 / OAuth（例如 GitHub） *(可选)*  
- JWT / Session 登录态管理 + 刷新机制  
- 用户资料管理（头像 / 昵称 / 简介 / 邮箱 / 注册时间等）  
- 管理员用户 / 普通用户 / 游客 等角色权限区分  

### 2. 博客文章系统  
- 博客文章的创建 / 编辑 / 删除 / 草稿 / 发布流程  
- 支持 Markdown（正文 + 标题 + 摘要 + 封面图 / 头图 /首图）  
- 文章分类 (Category)  
- 文章标签 (Tag)  
- 文章状态 (Draft / Published / Archived)  
- 支持图片、附件上传 (图床 / 本地存储 / OSS)  
- 文章列表 / 文章详情展示  
- 分页 / 排序 / 最新 / 推荐 / 热门文章列表  

### 3. 评论系统  
- 文章下评论 / 回复评论 (多级 comment)  
- 支持匿名 / 登录用户评论  
- 评论审核 / 管理 (管理员功能)  
- 点赞 / 点踩评论 (或者仅点赞)  
- 评论分页 /排序  

### 4. 文章分类 / 标签 / 归档 /导航结构  
- 分类视图 (按 category 浏览)  
- 标签视图 (按 tag 浏览)  
- 按年月归档 (Archive)  
- 导航菜单 / 首页 /分类/标签/归档/关于/联系 等基础页面  

### 5. 基础站点配置 / 全局信息管理  
- 站点标题 / 描述 /版权 /备案号 /社交媒体链接 /Meta 信息  
- SEO 相关字段 (title, description, slug/URL, canonical, etc.)  

### 6. 站点访问统计 & 阅读量统计  
- 文章阅读量 (views / hits) 统计  
- PV / UV / 基本访问统计 (可按文章 / 全站 / 时间区间)  

---

## 二、博客增强功能  

### 7. 搜索 & 全文检索  
- 站内搜索 (关键词搜索文章 / 标签 / 分类 /用户)  
- （高级）全文检索支持 (全文索引 + 高亮 + 分页) — 可选用 ES 或类似机制  

### 8. RSS / 订阅 / 邮件通知 /订阅功能  
- 提供 RSS feed (e.g. `/rss.xml`)  
- 支持用户通过邮箱订阅 (Email Subscription) *(可选)*  
- 新文章发布 / 评论 /回复时 Email 通知 / 站内通知  

### 9. 社交分享 / 社交媒体集成  
- 文章页分享按钮 (分享到 Twitter / 微信 /微博 /其它)  
- 社交媒体链接 (博客作者的社交媒体主页)  

### 10. 标签云 / 热门标签 /分类云 /标签/分类统计显示  
- 根据文章数 /访问量 /用户行为统计标签／分类热度  
- 标签云 / 分类云 /热门标签 /热门分类展示  

### 11. 相关文章 / 推荐文章 / 最近 /热门 /相似文章推荐  
- 根据标签 /分类 /关键词 /阅读行为推荐相关文章  
- “你可能还喜欢” / “相关推荐” / “热门文章”模块  

### 12. 自定义页面 & 静态页面支持 (About / 友情链接 / Contact / 简历 /项目展示)  
- About me / 个人简介页面  
- Contact /留言 / 联系方式 /社交媒体 /邮箱  
- 项目展示 / 作品集 (Portfolio) / 简历 (Resume) 页面  
- 友情链接 / 赞助页 /支持方式页面  

### 13. 响应式 / 多设备 /移动端适配 /暗黑模式 *(前端 + 后端配合支持)*  
- 为手机 /平板 /桌面适配 UI + 内容呈现  
- 支持暗黑 /浅色模式切换  

---

## 三、用户互动 & 社区功能  

### 14. 文章点赞 / 收藏 / 分享统计 / 用户收藏夹  
- 文章点赞／踩／收藏 (Favorite) 功能  
- 用户收藏夹 / 收藏列表  
- 分享计数 /点击统计  

### 15. 访客留言板 / Guestbook / 联系表单  
- 留言 /反馈 /建议提交机制  
- 后台审核 / 管理留言  

### 16. 站内通知 / 消息系统  
- 用户有新评论 /回复通知  
- 管理后台通知 (审核通知 /系统消息)  
- 可扩展为站内信系统 (Inbox /通知中心)  

### 17. 多人协作 / 多作者 /角色分工 (博主 / 编辑 /管理员) *(可选)*  
- 支持多个作者账号 /合作者 /团队博客  
- 每个作者拥有独立资料 /文章 /权限管理  
- 管理员可以管理所有内容 (文章 /用户 /评论 /留言)

## 四、创作者 /博主效率 & 管理工具  

### 18. 后台管理 / Admin 面板 (RESTful / Web UI)  
- 文章 / 分类 / 标签 /评论 /用户 /留言 管理界面  
- Dashboard: 站点统计 (文章数 / 用户数 /评论数 /访问统计)  
- 草稿管理 /已发布文章管理 /删除 /还原  

### 19. 自动 / 定时任务 /定时发布 /计划任务 (Scheduler)  
- 定时发布文章 (schedule publish)  
- 自动清理 /归档 /备份 /日志 rotation  

### 20. 草稿自动保存 / 编辑器支持 (草稿恢复)  
- 用户写文章时自动保存草稿 (本地 / 后端)  
- 支持恢复 /继续编辑  

### 21. 日志 / 审计 /系统监控 /健康检查 /错误追踪  
- 请求日志 /操作日志 /错误日志 /访问日志  
- 系统健康检查接口 (`/health`)  
- 数据库 /缓存 /外部依赖服务健康状态检查  
-（可选）异常报警 (邮件 / Slack /Webhook)  

### 22. 性能优化 & 缓存机制 /静态内容 /图片 /资源处理  
- 缓存热门文章 /热门标签 /导航 /分类 (Redis /内存)  
- 静态资源 /图片 /附件处理 (压缩 /缩略图 / CDN / OSS)  
- 图片 /附件 图床 支持

## 五、高级 / 可选 /“玩具”功能  

### 23. 全文搜索 + 智能搜索排序 /高亮 /搜索历史 /推荐  
- 使用 Elasticsearch 或类似方案实现全文检索  
- 支持关键词高亮、排序、分页  
- 搜索建议 /搜索历史 /热门搜索词  

### 24. 相关推荐 /智能推荐 /“猜你喜欢”机制  
- 根据用户浏览 /点赞 /收藏 /评论历史 + 文章内容 /标签 /分类 推荐相关文章  
- 推荐算法可以简单，也可以复杂 (协同过滤 /内容推荐)  

### 25. 个人知识图谱 / 文章关系图 /内容图谱 /主题图谱 (Graph / 可视化)  
- 构建标签／分类／文章之间的图结构 (Graph)  
- 提供接口 /数据支持给前端绘制知识图谱 /关系网络  

### 26. AI / 自动化功能 (可选)

- 自动生成文章封面 /缩略图 /配图建议  
- 自动打标签 /分类建议 (Tag /Category suggestion)  
- (可选) 自动评论 /推荐 /互动 (慎用)  

### 27. 实时 / 在线功能 / WebSocket /在线访客统计 /实时聊天 /简易社交功能 *(实验性质)*  
- 实时访客数 /在线用户统计
